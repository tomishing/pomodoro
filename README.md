# CLI Pomodoro Timer

This is a simple Pomodoro timer that runs in the terminal, written in JavaScript. The Pomodoro cycle consists of 25 minutes of focused work followed by a 5-minute break. After completing four Pomodoro cycles, you'll be prompted to decide whether to start another set of four cycles.

To stop the program at any time, press Ctrl+C.

Usage:

```sh
node pomodoro.js
```
